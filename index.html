스타일 센터

온라인 패션 쇼핑몰


    프로젝트 목표

    (1) 자신이 원하는 스타일의 옷들이 있는 쇼핑몰 사이트를 찾을려면 옮겨가면서 검색하는 불편함이 있었음
    (2) 웹으로 카테고리를 제시해서 선택한 카테고리를 보유한 쇼핑몰 사이트들을 제공하는 편의성을 제공하고자 함
    (3) 위와 같은 기능을 포함하고 기본 기능(로그인, 회원가입 등)이 있는 온라인 패션 쇼핑몰을 만들고자 함


    참여자 명단

  1. 이승영 : 백 엔드 개발 (크롤링), 중간 발표, 보고서 작성
  2. 이준규 : 백 엔드 (기능 구현)개발, 최종 발표
  3. 박서윤 : 프론트 엔드 개발 (웹 홈페이지 개발) 프로젝트 주제 발표

실제 기능 결과 UI


                    (1) 스타일 센터 홈페이지
                     남성 OR 여성을 선택 후 검색어에 쇼핑몰 이름이나 카테고리를 입력 후 검색

< (1-1) 접속 후 처음 화면 >

< (1-2) 쇼핑몰 이름이나 원하는 쇼핑몰의 특징 입력 >

< (1-3) 력 후 검색 결과 모습 >


                   (2) 회원가입, 로그인 페이지
                    아이디, 비밀번호, 이메일, 성별, 키, 상의사이즈, 하의사이즈를 입력 후 회원가입, 아이디와 비밀번호 입력 후 로그인

< (2-1) 회원가입 화면 >

< (2-2) 로그인 화면 >


                    (3) 남성, 여성 쇼핑몰 목록 페이지
                     쇼핑몰 목록에서 나이대, 스타일을 선택해서 설정 버튼을 눌러서 원하는 쇼핑몰 검색
                     ex) 20대와 러블리로 설정하면 (3-2) 화면처럼 쇼핑몰 결과가 나옴

< (3-1) 여성 쇼핑몰 카테고리 입력 화면, 남성 쇼핑몰도 같음 >

< (3-2) 카테고리를 입력 후 모습 >


                    (4) 구독 페이지
                     현재 로그인된 회원이 쇼핑몰로에서 구독 추가를 한 경우 (4-1)화면처럼 나옴
                     원하는 쇼핑몰의 (쇼핑몰로 이동) 버튼을 누르면 (4-2)처럼 해당 쇼핑몰로 이동

< (4-1) 회원 qwerty가 선택한 쇼핑몰 구독 목록 >


< (4-2) 구독 목록 중에 (데이빈)쇼핑몰을 선택한 결과 >


                    (5) 상품 목록 페이지
                     상품들을 나열해주는 페이지

< (5) 상품목록을 보여주는 결과 화면 >
